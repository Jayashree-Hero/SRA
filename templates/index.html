Instructions for Excel Sheet Format

The Excel sheet should be structured with columns such as S. No., Reg. Number, Section, Student Name, Subject code 1, Subject code 2... and so on. Ensure that column name is accurate and follows the exact order as mentioned. No columns should be left blank in between for correct insights.

The Excel sheet must not have any empty spaces in between the data. If found then the student will be considered as absent, this will affect the final outcome of the analysis or insight. The students who were absent can be marked as AA, AB or NA. Check the excel sheet once before uploading it in the website.

The Staff Excel sheet should be structured with columns such as Course Code, Course Name, Staff Name, Section. The columns must follow the same format and order as mentioned.
